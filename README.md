# Password Generator

A browser-based password generator built as a learning project.

Originally started as an exercise from SoftUni's JavaScript Fundamentals course, this project has since grown into a small work-in-progress that combines core JS concepts into a complete, functional page built from scratch.

**Images:** All PNG assets are free to use — downloaded from [freepik.com](https://www.freepik.com), designed by heisenberg_jr and improstudio.

> **Disclaimer:** For educational purposes only. All passwords are generated client-side and are never stored or transmitted.
